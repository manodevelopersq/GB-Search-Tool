GB Search Tool

To edit the list, enter the name of the user or the word you want to block in their respective areas. It must be written exactly as it is on the website. Each line represents an element, so break the line for each term and then save.

If you just put the word you want to block, it will block any variation, for example "program" will block "programmer" too, if you want to block an exact word, put "[EXACT]" before the word you want to block, like "[EXACT]program", that way, the word "programmer" will not be blocked
